# Venux Studio

Scaffold Next.js project and design system for Venus Bijoux e commerce.

Structure includes pages, layouts, components, contexts, hooks, services, styles, design-system and public assets. The theme and tokens are implemented in `design-system/themes.ts` and `design-system/tokens.json`.

Run locally

```bash
cd venux-studio
npm install
npm run dev
```

This scaffold includes light and dark coffee themed tokens and starter components. Continue by implementing pages, admin panel and backend integration.
